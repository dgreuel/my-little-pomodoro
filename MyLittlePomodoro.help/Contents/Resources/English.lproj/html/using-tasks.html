My Little Pomodoro Help

Tasks

Keeping track of your tasks while working is an important part of The Pomodoro Technique, and My Little Pomodoro aims to make task management straightforward and puts it entirely in the hands of the user. Using My Little Pomodoro's task management is also completely optional.

Note: If you do opt to use the built-in task management features, adding a new task is simple, but there are a few things to keep in mind:
  • A task must be selected for any statistics to be recorded.
  • It is up to you to add or select a new task once you are finished with the previous task.
  • Adding, removing, or selecting a new task while the timer is stopped will end the current timer.
    • If you follow the Pomodoro Technique exactly as it's prescribed, once the timer begins, it has to ring. The exception to this is if "you finish an activity in the first five minutes of the Pomodoro and you feel like the task was actually already finished during the previous Pomodoro and revision wouldn’t be worthwhile" (The Pomodoro Book, pages 7-8), in which case you can end the Pomodoro early and not include it in the Pomodoro count. This is why My Little Pomodoro ends the current timer if you stop the timer and try to change/add/remove tasks.

To add a task:

  1. Add a task.
    • click on the Add Task button (the + button) in the lower left hand corner of the application window
    • or choose Tasks > Add Task
    • or type ⌘T
  2. Type a name for the new task.
  4. Hit the return key

After adding a task, you can start the timer. A task must be selected for any statistics to be kept about the task.

Once you have completed a task, click on the checkbox to the left of the task name to mark it as completed.

To remove a task:

  1. Select the task that you want to remove.
  2. Remove the task.
    • click on the Remove Task button (the - button) in the lower left hand corner of the application window
    • or choose Tasks > Remove Task
    • or type ⌘⌫

You can also remove all completed tasks.

To remove all completed tasks:

  • choose Tasks > Remove Completed Tasks
  • or type ⌥⌘⌫

Stats

You will notice that as you work (and a task is selected), some basic statistics will be displayed for the selected task. You will see the date the task was created, the number of completed Pomodoros, the total number of completed Pomodoro minutes, and the total number of stops for that task.
