Alarm Notifications

You will probably be doing work in other applications while using My Little Pomodoro, so the time left for the current timer is displayed on the My Little Pomodoro dock icon.

The My Little Pomodoro dock icon will bounce if you are using another application when the timer finishes.

You can also choose to be notified via Growl, if you have it installed (http://www.growl.info).

To change these notifications:
