My Little Pomodoro Help

Getting Started

To get started with My Little Pomodoro, select the type of timer you want (Pomodoro, short break, long break), and that's it! The timer will begin counting down.

At this point, you can stop the timer by clicking the Stop button. A stopped timer can either be resumed by clicking on the Start button, or reset by clicking on the Reset button.

Note: You can also use the keyboard shortcut ⌘S to stop or start a timer. ⌘R resets a stopped timer.

When the timer finishes, you will be alerted by audio alarms and notifications (which are customizable). After the timer finishes, you can again select the type of timer you would like.

You can also use My Little Pomodoro to keep track of the tasks that you are working on. See Tasks and Stats for more information.
